Turf.js Features Showcase

This map demonstrates various features of Turf.js, a geospatial analysis library.

  • Markers: Simple markers are added to the map.
  • Distance: Calculate and display the distance between two points.
  • Buffer: Create and display a buffer around a point.
  • Centroid: Find and display the centroid of a set of points.
  • Random Points: Generate and display random points within a bounding box.
